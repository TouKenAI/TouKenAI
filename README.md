## 📊 实战案例（代表性交付场景）

> 以下为投肯智能在「企业 AI 私有化落地 × 信创合规」方向的**代表性交付场景**，综合多个内网离线项目抽象而成，用于展示完整能力链路；具体客户名称与指标以实际签约项目为准。

### ⭐ 某大型装备制造企业 · 本地局域网离线 AI 助手
**场景痛点**
- 工厂 20 年积累的研发 / 工艺 / 质检文档散落（图纸、工艺卡、SOP、质检记录），工程师查一份规范平均耗时 30 分钟；
- 因保密合规要求**全厂禁止上外网 / 公有云**，无法使用任何在线大模型。

**投肯方案（FDE 现场部署，零出网）**
1. **大模型私有化**：厂内服务器 vLLM 部署 Qwen-72B（INT4 量化），纯内网推理，数据不出厂；
2. **知识库 RAG**：LlamaIndex + 向量库，灌入 PDF 工艺卡 / Word SOP / Excel 质检表 / 扫描图纸 OCR，检索增强；
3. **Agent 软件**：LangGraph 多 Agent（文档检索 + 工艺问答 + 报表生成），接企业微信 / 内网 Web 前端，自然语言提问；
4. **办公文档自动化**：会议纪要 / 变更单自动生成标准 Word / Excel，对接 OA。

**典型成效（示意区间）**
- 规范查询耗时 **30 分钟 → < 1 分钟**（约 30× 提速）
- 新人上手周期 **缩短 ~60%**
- 全厂 **零数据出网**，通过保密自查 / 等保要求
- 知识库覆盖 **50 万+ 页** 历史文档
